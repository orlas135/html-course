Etiquetas semánticas estructurales

Es una etiqueta contenedora que no tiene semántica
Header - Cabecera de un sitio o de una sección
Main - Define la sección principal del documento, solo puede existir una vez por documento HTML
Footer - Pie de página de un sitio WEB o una sección
Nav - Representá un menú de navegación
Article - Representa una sección de autocontenido (que por sí sola se explica)
Aside - Representa un contenido complementario o secundario, también se utiliza para meter publicidad
Sectin - Representa una sección de contenido genérico
Address - Representa una dirección física

Etiquetas de bloque y etiquetas de línea en HTML

La etiqueta de bloque por excelencia es Div. Las etiquetas de bloque son aquellas que generan saltos de línea con sus etiquetas hermanas
La etiquera de línea por excelente es span. Una etiqueta de línea solo ocupa el espacio necesario que tiene su contenido
